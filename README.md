## 🎯 Goal

Practice HTML & CSS by building the dashboard for our **Full-Stack** project — a student platform for organizing and sharing class notes.

You'll be styling a pre-built HTML file using CSS fundamentals:
- Box model (padding, margin, borders)
- Flexbox layout
- Selectors, colors, fonts, and sizing

| File | Description |
|---|---|

| `styles.css` | ⚙️ Your job — fill in the CSS |
| `iNoted-sample-dashboard.png` | 🌃 Reference image of the finished dashboard |

## 🧱 Dashboard Sections to Style

Your dashboard should have these four core sections:

1. **Sidebar** — navigation links + user info at the bottom
2. **Class Card** — one card showing an enrolled class (we'll make more with React later)
3. **Upload Notes** — a small form to upload a file
4. **Recent Notes** — a list of recently shared notes

## 🚀 Getting Started

### Option A — Download the files
1. Download `index.html` and `styles.css` from this branch
2. Create a new folder on your computer (e.g. `project-dashboard`)
3. Move both files into that folder
4. Open the folder in **VS Code**
5. Open `index.html` with **Live Server** to preview in the browser

### Option B — Fork and Clone the repository
```bash
1. Fork the repo on Github
git clone <repo-url>
cd <repo-folder>
git checkout Day2
```
Then open the folder in VS Code and launch with Live Server.

---

## ✏️ Instructions

1. Open `styles.css`. The font, reset, and body are already done for you
2. Each CSS selector has a comment above it explaining what properties to add
3. Fill in each empty selector `{ }` with the right CSS
4. Save and check your browser after every few changes
5. Use the sample image above as your reference for how it should look

---

## 💡 Tips

- **Follow the comments** in `styles.css` if you're unsure what to write, they guide you property by property
- **Don't follow them** if you want a challenge, style it completely your own way
- Open the sample image and your browser side by side to compare as you go
- You only have **30 minutes** so focus on the CSS, don't get stuck on the HTML

---

## ⚡ Advanced (Optional)

If you finish early or want a bigger challenge:
- Edit `index.html` to use your own content
- Try changing the color theme
- Add a new section or modify the layout

> ⚠️ Only edit the HTML structure (adding/removing sections) if you're comfortable with it. Stick to CSS if time is limited.

---

## 📌 Key Concepts You'll Practice

- `display: flex` and flex properties (`flex-direction`, `align-items`, `justify-content`, `gap`, `flex: 1`)
- `margin-top: auto` to push elements to the bottom
- `border-radius`, `border`, `padding`, `background-color`
- Styling form elements (`input`, `select`, `button`, `label`)
- Hover states with `:hover`
- Hiding and replacing the default file input
